# KundliAI Marketplace

A Claude plugin marketplace for authentic Vedic astrology tools.

## Plugins available

### `kundli-ai`
Six skills for Vedic Kundli reading inside Claude:
- **kundli-basic-report** — 1,500-word starter reading
- **kundli-detailed-report** — 2,500-word deep reading with marriage, health, and 2025–26 month-by-month forecast
- **kundli-premium-report** — 3,000-word personal kundli + Kundli Milan compatibility
- **kundli-milan-compatibility** — standalone Ashtakoota / Guna Milan score
- **monthly-horoscope** — personalised month-ahead forecast
- **daily-rashifal** — short daily horoscope by moon sign

## Install

In Claude Code or Cowork, add this marketplace:

```
/plugin marketplace add r-balaganapathi/kundliai-marketplace
```

Then install the plugin:

```
/plugin install kundli-ai@kundliai-marketplace
```

Or download the prebuilt `.plugin` file from [Releases](../../releases) and install via the Cowork UI.
